# java-shareit

Backend-сервис для бронирования вещей между пользователями. Проект реализован в виде монолитного REST-приложения на современном стеке Spring Boot. Позволяет пользователям добавлять вещи, управлять их доступностью, бронировать на определённые даты, а также оставлять запросы на те вещи, которых нет в системе.

## 🛠️ Стек технологий
Java 21, Spring Boot 3.3.2, Spring Data JPA, Hibernate, PostgreSQL 16.1, H2 (тесты), Maven, Docker, Lombok 1.18.30, Bean Validation, JUnit 5, JaCoCo 0.8.12, Checkstyle 10.17.0, SpotBugs 4.8.5.0

## 🚀 Особенности проекта

👤 Пользователи
- CRUD-операции
- Проверка уникальности email

📦 Вещи
- Добавление и управление вещами
- Возможность указать доступность вещи для бронирования

📅 Бронирование

- Создание бронирования на определённый период
- Запрет бронирования собственной вещи
- Проверка пересечения дат бронирований
- Подтверждение или отклонение бронирования владельцем
- Фильтрация бронирований по статусам: ALL, CURRENT, PAST, FUTURE, WAITING, REJECTED
- Для каждой вещи можно получить последнее и следующее бронирование

💬 Запросы вещей
- Пользователи могут создавать запросы на вещи, которых ещё нет
- Возможность отвечать на запросы, добавляя новые вещи

## 🏗️ Архитектура и технические решения

- Многослойная архитектура: Controller → Service → Repository (Spring Data JPA)
- Использование DTO для изоляции API от доменной модели
- Глобальная обработка исключений через @ControllerAdvice
- Управление транзакциями с @Transactional
- Логирование с SLF4J (настроена кодировка UTF-8)
- Две базы данных: PostgreSQL для основного запуска, H2 для тестов
- Покрытие кода тестами (JUnit 5) с отчётами JaCoCo (>80%)
- Статический анализ кода: Checkstyle, SpotBugs
- Maven-профили: check (запуск анализаторов) и coverage (генерация отчёта JaCoCo)

## ▶️ Системные требования и запуск
**Требования**
Java 21  
Maven 3.9+  
Docker (для запуска PostgreSQL)

**Запуск**
```bash
# Клонирование репозитория
git clone https://github.com/tubashov/java-shareit.git
cd java-shareit

# Запуск PostgreSQL в Docker (будет доступна на порту 5432, логин/пароль: postgres/postgres)
docker-compose up -d

# Сборка и запуск приложения
mvn clean install
mvn spring-boot:run
# Приложение будет доступно по адресу: http://localhost:8080
